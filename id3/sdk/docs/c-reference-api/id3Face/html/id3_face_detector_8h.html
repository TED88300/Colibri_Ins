◆ id3FaceDetector_GetNmsIouThreshold()

ID3FACE_C_API int ID3_CDECL id3FaceDetector_GetNmsIouThreshold ( ID3_FACE_DETECTOR  hFaceDetector,
int *  nmsIouThreshold 
)

Gets the non-maximum suppression (NMS) intersection-over-union (IOU) threshold. Default value is 40. Range is [0;100]. Setting a high threshold allows to detect more overlapping faces which can be useful in a multi-face secenario. On the contrary, in a portrait scenario, a low NMS IOU threshold should be preferred.

Parameters
hFaceDetector[in] Handle to the FaceDetector object.
nmsIouThreshold[out] Non-maximum suppression (NMS) intersection-over-union (IOU) threshold. Default value is 40. Range is [0;100]. Setting a high threshold allows to detect more overlapping faces which can be useful in a multi-face secenario. On the contrary, in a portrait scenario, a low NMS IOU threshold should be preferred.
Returns
An error code. See id3FaceError file for the list of possible error codes.
Return values
id3FaceError_SuccessThe function succeeded.

◆ id3FaceDetector_TrackFaces()

ID3FACE_C_API int ID3_CDECL id3FaceDetector_TrackFaces ( ID3_FACE_DETECTOR  hFaceDetector,
ID3_FACE_IMAGE  hImage,
ID3_DETECTED_FACE_LIST  hDetectedFaceList 
)

Tracks faces in an image and update their info in a DetectedFaceList object. The algorithm searches for faces in the range [16px;512px]. If the image is too large to fit this range, one must resize it before the detection process. In a realtime process, one must use this function instead of "detect" to keep the face IDs stable in time.

Parameters
hFaceDetector[in] Handle to the FaceDetector object.
hImage[in] Source image to process.
hDetectedFaceList[in] List of detected faces.
Returns
An error code. See id3FaceError file for the list of possible error codes.
Return values
id3FaceError_SuccessThe function succeeded.

Retrieves a license file using id3 account credentials and a product reference.

Parameters

◆ id3FaceLicense_LoadLicense()

ID3FACE_C_API int ID3_CDECL id3FaceLicense_LoadLicense ( const char *  licensePath,
void *  optionalParameter 
)

Loads and checks a license from a file.

This function does two things:

  • First it loads the license from a file using the given path
  • Then it checks the validity of the license regarding the host it is called on (Windows, Linux, Android ...)

Calling this function is required to get access to the license member getters.

The optional parameter is required on Android and must be the JNIEnv* pointer casted as a void*. It can be set as NULL for other devices.

Parameters
licensePath[in] Absolute or relative path to the license file.
optionalParameter[in] Only used on Android: JNIEnv* pointer casted as a void*. Ignored for other devices.
Returns
An error code. See id3FaceError file for the list of possible error codes.
Return values
id3FaceError_SuccessThe function succeeded.

Loads and checks a license from a buffer.

This function does two things:

  • First it loads the license from a buffer
  • Then it checks the validity of the license regarding the host it is called on (Windows, Linux, Android ...)

Calling one of the license loading function is required to get access to all the license member getters.

Parameters

optionalParameter[in] Only used on Android: JNIEnv* pointer casted as a void*. Ignored for other devices.

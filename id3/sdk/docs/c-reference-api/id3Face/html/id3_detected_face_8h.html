ID3FACE_C_API int ID3_CDECL id3DetectedFace_GetPortraitBounds ( ID3_DETECTED_FACE  hDetectedFace,
float  eyeImageWidthRatio,
float  eyeImageHeightRatio,
float  imageRatio,
id3FaceRectangle *  sPortraitBounds 
)

Gets the portrait bounds of the detected face.

Parameters
hDetectedFace[in] Handle to the DetectedFace object.
eyeImageWidthRatio[in] Ratio between eye distance and image width. Must be in the range ]0;1[. Default recommended value is 0.25.
eyeImageHeightRatio[in] Ratio between eye distance to top and image height. Must be in the range ]0;1[. Default recommended value is 0.45.
imageRatio[in] Ratio between image height and image width. Default recommended value is 1.33 (4/3).
sPortraitBounds[out] Portrait bounds of the detected face.
Returns
An error code. See id3FaceError file for the list of possible error codes.
Return values
id3FaceError_SuccessThe function succeeded.

ID3_DETECTED_FACE  hDetectedFace,
int *  quality 
)

Computes the quality of a detected face. Here, quality expresses the match capability of an image: a high quality image will generate less match errors (false acceptance or false rejection) than a low quality one.

Parameters
hFaceEncoder[in] Handle to the FaceEncoder object.
hImage[in] Source image to process.
hDetectedFace[in] Detected face to process.
quality[out] Estimated quality of the detected face. Range is [0:100].
Returns
An error code. See id3FaceError file for the list of possible error codes.
Return values
id3FaceError_SuccessThe function succeeded.
